# Garbage Trucks Frontend

PWA приложение на Next.js для управления маршрутами мусоровозов.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно на http://localhost:6000

## Сборка

```bash
npm run build
npm start
```

## Возможности

- ✅ PWA (работает офлайн)
- ✅ Список водителей
- ✅ Маршруты с точками сбора
- ✅ Обновление статусов в реальном времени
- ✅ Адаптивный дизайн
